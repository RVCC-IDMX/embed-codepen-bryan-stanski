Pen: Multi-layered Parallax Illustration

by Patryk Zabielski

This pen shows how you can achieve a Multi-layered illustration just using css & javascript. By splitting your image into multiple image files you can easily layer the images in css and then using javascript you can make an event for scrolling that causes the layers to form.
